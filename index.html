**GitHub Tools Hub** ====================== صفحة جلب أدوات احترافية باستخدام GitHub API والتقنيات الحديثة. **محتويات** ----------- * HTML: لتنظيم مكونات الصفحة * CSS: باستخدام Tailwind CSS لتخصيص تصميم الصفحة * JavaScript: لاستخدام API GitHub والجلب الأدوات **HTML** ------ GitHub Tools Hub

GitHub Tools Hub

بحث عن أدوات GitHub

**JavaScript** ------------- document.addEventListener("DOMContentLoaded", function () { const form = document.querySelector("form"); const searchInput = document.querySelector("input"); const resultsDiv = document.querySelector("#results"); form.addEventListener("submit", function (e) { e.preventDefault(); const searchQuery = encodeURIComponent(searchInput.value); fetch(`https://api.github.com/search/repositories?q=${searchQuery}`) .then(response => response.json()) .then(data => { const results = data.items; const resultsHtml = results.map(result => `

${result.name}

${result.description}

المؤلف: ${result.owner.login}

عدد التكلمات: ${result.stargazers_count}

`).join(""); resultsDiv.innerHTML = resultsHtml; }) .catch(error => console.error("حدث خطأ:", error)); }); }); **تعلمك كيفية عمل هذا المشروع** 1. **إنشاء مشروع جدول باستخدام HTML**: استخدم HTML للإنشاء البنية الأساسية لمشروع جدول. 2. **تضمين CSS Tailwind**: استخدم Tailwind CSS لتخصيص تصميم المشروع. 3. **إنشاء معاينة باستخدام JavaScript**: استخدم Fetch API لجلب البيانات من GitHub API. 4. **حساب نتائج**: استخدم JavaScript للتعرف على البيانات المستلمة من API وإنشاء HTML معاينة للأدوات. **أخبار ومواضيع مرتبطة** * [تكنولوجيا المعلومات](https://ar.wikipedia.org/wiki/تكنولوجيا_المعلومات) * [GitHub](https://ar.wikipedia.org/wiki/GitHub) * [Tailwind CSS](https://ar.wikipedia.org/wiki/Tailwind_CSS)
